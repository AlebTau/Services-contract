# Юридичні документи ФОП Бойцев О.Д.

Цей репозиторій містить актуальні версії публічних договорів та умов надання послуг (Terms of Service).

## Документи:
* [**Публічна оферта про надання ІТ-послуг**](./OFFER.md) — основний договір для надання послуг із розробки, конфігурування та міграції ПЗ.
* [**Політика конфіденційності**](./PRIVACY.md) — умови обробки даних (якщо плануєте додавати пізніше).

## Як це працює:
Відповідно до законодавства України, оплата рахунку (інвойсу) із посиланням на цей репозиторій є акцептом умов договору. 

**Важливо:** Історія змін (Git History) цього репозиторію дозволяє перевірити редакцію договору, що діяла на момент здійснення оплати.
